## 1. HS Code 개요

### 1.1 정의
**HS Code (Harmonized System Code)**는 세계관세기구(WCO)가 제정한 국제통일상품분류체계입니다.

### 1.2 구조
```
HS Code 구조: XX.XX.XX.XXXX
              ││ ││ ││ ││││
              ││ ││ ││ ││││
              ││ ││ ││ │└└└─ 국가별 세분류 (4자리)
              ││ ││ ││ └──── 국가별 세분류 (3자리)
              ││ ││ └└────── 호 (2자리) - 국제 공통
              ││ └└────────── 류 (2자리) - 국제 공통
              └└──────────── 부 (Section) - 로마숫자
```

- **처음 6자리**: 국제적으로 통일된 코드 (전 세계 공통)
- **7-10자리**: 각 국가가 자율적으로 세분화 (한국은 HSK Code)

### 1.3 한국의 HSK Code
- **10자리 체계**: 관세청에서 관리
- **용도**: 수출입 통관, 관세 부과, 무역 통계

## 4. HS Code 적용 실무

### 4.1 HS Code 결정 프로세스
```
1단계: 제품의 기본 특성 파악
   ↓
2단계: 해당하는 류(Chapter) 찾기
   ↓
3단계: 세부 호(Heading) 확인
   ↓
4단계: 국가별 세분류 확인 (HSK)
   ↓
5단계: 관세청 유권해석 확인
   ↓
6단계: 최종 HS Code 확정
```

### 4.2 HS Code 분류 시 고려사항

#### 상태별 분류
- **신선 (Fresh)**: 냉장되지 않은 상태
- **냉장 (Chilled)**: 0°C ~ 4°C 정도로 냉각
- **냉동 (Frozen)**: -18°C 이하로 동결
- **건조 (Dried)**: 수분을 제거한 상태
- **조제 (Prepared)**: 가공 처리한 것

#### 가공 정도별 분류
- **1차 가공**: 세척, 절단, 박피 등
- **2차 가공**: 조리, 조미, 혼합 등
- **3차 가공**: 포장, 멸균, 통조림 등

#### 포장 단위별 분류
- **소매 포장 (Retail packaging)**: 직접 소비자에게 판매
- **업소용 포장 (Bulk packaging)**: 대용량 포장

### 4.3 혼합 제품의 분류 원칙

#### 본질적 특성 원칙 (Essential Character)
혼합 제품은 전체 제품의 본질적 특성을 부여하는 재료의 HS Code를 따릅니다.

예시:
- **딸기잼**: 설탕이 많지만 본질은 딸기 → 2007류 (과실 조제품)
- **초콜릿 비스킷**: 비스킷이 본질 → 1905류 (베이커리 제품)

### 4.4 HS Code 조회 방법

#### 온라인 조회
1. **관세청 관세법령정보포털** (https://unipass.customs.go.kr)
   - HS Code 검색
   - 품목분류 사전심사 신청

2. **관세청 통관포털** (https://portal.customs.go.kr)
   - 수출입 통관 정보
   - 품목분류 상담

3. **WCO 공식 사이트** (https://www.wcoomd.org)
   - 국제 표준 HS Code

#### 유권해석 신청
- **목적**: 불명확한 제품의 공식적인 HS Code 판정
- **신청처**: 관세청 품목분류과
- **소요 기간**: 약 30일
- **효력**: 법적 구속력 있음

## 5. HS Code와 관세율

### 5.1 관세율 종류
- **기본세율**: WTO 회원국에 적용되는 기본 세율
- **협정세율**: FTA 협정에 따른 특혜 세율
- **할당관세**: 일정 물량에 대해 낮은 세율 적용
- **긴급관세**: 국내 산업 보호를 위한 추가 관세

### 5.2 주요 식품 관세율 (예시)

```
HS Code    품목              기본세율    한-미 FTA    한-EU FTA
0201.10    소고기(냉장)      40%        단계철폐     단계철폐
0303.14    참치(냉동)        10%        0%          0%
0406.90    치즈              36%        단계철폐     단계철폐
0709.60    고추(신선)        270%       유지        유지
1006.30    쌀(정미)          513%       유지        유지
1901.90    분유              176%       단계철폐     단계철폐
2103.90    고추장            45%        0%          0%
```

### 5.3 FTA 원산지 증명
HS Code는 FTA 원산지 판정 기준이 됩니다.

- **직접운송 원칙**: 협정국에서 직접 운송
- **역내부가가치 기준**: 일정 비율 이상 역내 생산
- **세번변경 기준**: HS Code 6단위 변경

### 6.1 테이블 구조 예시

#### hs_codes 테이블
```sql
CREATE TABLE hs_codes (
    id SERIAL PRIMARY KEY,
    hs_code VARCHAR(10) NOT NULL UNIQUE,
    hs_code_6 VARCHAR(6) NOT NULL,  -- 국제 공통 코드
    chapter VARCHAR(2) NOT NULL,     -- 류

    description_ko TEXT NOT NULL,    -- 한글 설명
    description_en TEXT,             -- 영문 설명
    parent_code VARCHAR(10),         -- 상위 코드
    level INTEGER,                   -- 분류 레벨 (1-5)
    is_leaf BOOLEAN DEFAULT FALSE,   -- 최하위 코드 여부
    effective_date DATE,             -- 시행일
    end_date DATE,                   -- 종료일
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 인덱스
CREATE INDEX idx_hs_codes_code ON hs_codes(hs_code);
CREATE INDEX idx_hs_codes_6 ON hs_codes(hs_code_6);
CREATE INDEX idx_hs_codes_chapter ON hs_codes(chapter);
```

#### tariff_rates 테이블
```sql
CREATE TABLE tariff_rates (
    id SERIAL PRIMARY KEY,
    hs_code VARCHAR(10) NOT NULL,
    country_code VARCHAR(2) NOT NULL,  -- 수출국
    rate_type VARCHAR(20) NOT NULL,    -- 'basic', 'fta', 'quota'
    tariff_rate DECIMAL(5,2),          -- 관세율 (%)
    fta_name VARCHAR(50),              -- FTA 협정명
    effective_date DATE NOT NULL,
    end_date DATE,
    FOREIGN KEY (hs_code) REFERENCES hs_codes(hs_code)
);
```

#### product_hs_mapping 테이블
```sql
CREATE TABLE product_hs_mapping (
    id SERIAL PRIMARY KEY,
    product_id INTEGER NOT NULL,
    hs_code VARCHAR(10) NOT NULL,
    is_primary BOOLEAN DEFAULT TRUE,  -- 주요 HS Code
    confidence_score DECIMAL(3,2),    -- 분류 정확도 (0.00-1.00)
    verified_by INTEGER,              -- 검증자 ID
    verified_at TIMESTAMP,
    notes TEXT,
    FOREIGN KEY (hs_code) REFERENCES hs_codes(hs_code),
    FOREIGN KEY (product_id) REFERENCES products(id)
);
```

## 7. HS Code 자동 분류 (AI 활용)

### 7.1 분류 알고리즘
```python
def classify_hs_code(product):
    """
    제품 정보를 기반으로 HS Code를 자동 분류
    """
    # 1. 키워드 기반 초기 필터링
    keywords = extract_keywords(product.name, product.description)
    candidate_codes = filter_by_keywords(keywords)
    
    # 2. 제품 속성 분석
    attributes = {
        'category': product.category,
        'material': product.material,
        'processing_level': product.processing_level,
        'state': product.state,  # fresh, frozen, dried, etc.
        'packaging': product.packaging
    }
    
    # 3. ML 모델 예측
    ml_prediction = ml_model.predict(attributes)
    
    # 4. 규칙 기반 검증
    validated_code = validate_with_rules(ml_prediction, attributes)
    
    # 5. 신뢰도 점수 계산
    confidence = calculate_confidence(validated_code, product)
    
    return {
        'hs_code': validated_code,
        'confidence': confidence,
        'alternatives': candidate_codes[:5]
    }
```

## 8. 국제 규정 및 컴플라이언스

### 8.1 수출입 규제 품목
일부 식품은 특별 허가가 필요합니다:

- **검역 대상**: 육류, 식물, 과일
- **허가 필요**: 건강기능식품, 알코올
- **금지 품목**: 멸종위기 동식물

### 8.2 라벨링 요구사항
수출 시 HS Code를 포함한 정보 표시:

- **원산지 표시**: Made in Korea
- **HS Code 표시**: 선적 서류에 명시
- **성분 표시**: 수입국 언어로 표시
- **알레르기 정보**: 주요 알레르기 유발 물질

### 8.3 증명서 및 서류
- **원산지 증명서** (Certificate of Origin)
- **위생증명서** (Health Certificate)
- **검역증명서** (Phytosanitary Certificate)
- **할랄 인증서** (Halal Certificate) - 이슬람 국가
- **코셔 인증서** (Kosher Certificate) - 유대교

### 9.1 예제 1: 김치 수출
```
제품: 배추김치 (냉장)
HS Code: 2005.99.1000

분류 근거:
- 제20류: 채소 조제품
- 2005: 기타 채소 (냉동하지 않은 것)
- 2005.99: 기타
- 2005.99.1000: 배추김치

관세율 (미국 수출):
- 기본세율: Free (0%)
- 한-미 FTA: 0%

필요 서류:
- 원산지 증명서
- 위생증명서
- 상업송장
```

### 9.2 예제 2: 홍삼 엑스 수출
```
제품: 홍삼 농축액 (병포장)
HS Code: 2106.90.5100

분류 근거:
- 제21류: 기타 조제 식료품
- 2106: 기타 조제 식료품
- 2106.90: 기타
- 2106.90.5100: 홍삼 엑스

관세율 (중국 수출):
- 기본세율: 10%
```
