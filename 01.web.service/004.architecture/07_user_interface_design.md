# 사용자 화면 설계 (Phase 1 MVP)

## 문서 정보
- 작성일: 2025-11-19
- 버전: 1.0 (Phase 1 MVP)
- 목적: 사용자 화면 구조 및 UI/UX 정의

---

## 1. 화면 구조 개요

### 1.1 사이트맵

```
SpicyJump
├── 홈 (/)
├── 로그인 (/login)
├── 회원가입 (/register)
├── 상품
│   ├── 상품 목록 (/products)
│   └── 상품 상세 (/products/:id)
├── 장바구니 (/cart)
├── 주문
│   ├── 주문하기 (/checkout)
│   ├── 주문 목록 (/orders)
│   └── 주문 상세 (/orders/:id)
├── 결제
│   ├── 결제하기 (/payment)
│   ├── 결제 완료 (/payment/success)
│   └── 결제 실패 (/payment/failed)
├── 마이페이지 (/mypage)
│   ├── 프로필 (/mypage/profile)
│   ├── 주문 내역 (/mypage/orders)
│   └── 리뷰 관리 (/mypage/reviews)
└── 판매자 (role: seller)
    ├── 대시보드 (/seller/dashboard)
    ├── 상품 관리 (/seller/products)
    └── 주문 관리 (/seller/orders)
```

## 2. 공통 레이아웃

### 2.1 전체 레이아웃 구조

```
┌─────────────────────────────────────────────────────────┐
│                      Header                             │
│  Logo | 카테고리 | 검색 | 장바구니 | 로그인/회원가입   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                    Main Content                         │
│                                                         │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                      Footer                             │
│  회사정보 | 이용약관 | 개인정보처리방침                  │
└─────────────────────────────────────────────────────────┘
```

### 2.2 반응형 브레이크포인트

```css
/* Mobile First */
/* Extra Small (Mobile) */
@media (min-width: 0px) { ... }

/* Small (Mobile Landscape) */
@media (min-width: 576px) { ... }

/* Medium (Tablet) */
@media (min-width: 768px) { ... }

/* Large (Desktop) */
@media (min-width: 992px) { ... }

/* Extra Large (Wide Desktop) */
@media (min-width: 1200px) { ... }
```

### 2.3 Header 컴포넌트

#### Desktop (1200px+)
```
┌────────────────────────────────────────────────────────────┐
│  [Logo]  카테고리▼  [검색창____________] 🛒 로그인 회원가입  │
└────────────────────────────────────────────────────────────┘
```

#### Mobile (< 768px)
```
┌────────────────────────────────┐
│  ☰  [Logo]        🛒  👤       │
├────────────────────────────────┤
│  [검색창__________________]    │
└────────────────────────────────┘
```

#### Header 요소
- **로고**: 클릭 시 홈으로 이동
- **카테고리 메뉴**: Dropdown 형태
- **검색창**: 상품명 검색
- **장바구니 아이콘**: 장바구니 수량 뱃지 표시
- **로그인/회원가입**: 로그인 상태 시 "마이페이지"로 변경
- **언어 선택**: 🌐 (한국어/English)

## 3. 주요 화면 설계

### 3.1 홈 화면 (S-001)

#### 화면 구성 (Desktop)
```
┌────────────────────────────────────────────────────────┐
│                     Hero Banner                        │
│   "Welcome to SpicyJump - Korean Food Marketplace"    │
│                  [Browse Products]                     │
├────────────────────────────────────────────────────────┤
│                  카테고리 목록                          │
│  [김치/반찬]  [장류/양념]  [면류]  [과자/간식]         │
├────────────────────────────────────────────────────────┤
│                   신상품 (New Arrivals)                │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                 │
│  │Image│  │Image│  │Image│  │Image│                 │
│  │김치  │  │고추장│  │라면  │  │김밥  │                 │
│  │$15  │  │$12  │  │$8   │  │$10  │                 │
│  │★4.5 │  │★4.8 │  │★4.7 │  │★4.3 │                 │
│  └─────┘  └─────┘  └─────┘  └─────┘                 │
├────────────────────────────────────────────────────────┤
│                   베스트 상품 (Best Sellers)            │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                 │
│  │     │  │     │  │     │  │     │                 │
│  └─────┘  └─────┘  └─────┘  └─────┘                 │
└────────────────────────────────────────────────────────┘
```

#### 화면 요소
1. **Hero Banner**
   - 배너 이미지 (Carousel)
   - CTA 버튼: "Browse Products"
   
2. **카테고리 섹션**
   - 주요 카테고리 4-6개
   - 아이콘 + 카테고리명
   - 클릭 시 해당 카테고리 상품 목록으로 이동

3. **신상품 섹션**
   - 최근 등록된 상품 8개
   - Horizontal Scroll (Mobile)
   - Grid Layout (Desktop)

4. **베스트 상품 섹션**
   - 판매량 또는 평점 기준 상위 8개

1. **Breadcrumb**: 홈 > 상품 > 카테고리명

   - 가격대
   - 평점
   - [필터 적용] 버튼
3. **상품 목록**
   - 정렬 옵션: 최신순, 가격 낮은순, 가격 높은순, 평점순
   - View 옵션: Grid / List
   - 상품 카드 (Grid: 4개/row, Mobile: 2개/row)
4. **Pagination**: 페이지 번호 버튼
