# 관리자 화면 설계 (Phase 1 MVP)

## 문서 정보
- 작성일: 2025-11-19
- 버전: 1.0 (Phase 1 MVP)
- 목적: 관리자/판매자 화면 구조 및 기능 정의

---

## 1. 화면 구조 개요

### 1.1 관리자 사이트맵

```
Admin Dashboard
├── 대시보드 (/admin/dashboard)
├── 회원 관리 (/admin/users)
│   ├── 회원 목록
│   ├── 회원 상세
│   └── 회원 등급 관리
├── 상품 관리 (/admin/products)
│   ├── 상품 목록
│   ├── 상품 상세/수정
│   └── 카테고리 관리
├── 주문 관리 (/admin/orders)
│   ├── 주문 목록
│   └── 주문 상세
├── 결제 관리 (/admin/payments)
│   ├── 결제 내역
│   └── 환불 처리
├── 리뷰 관리 (/admin/reviews)
│   ├── 리뷰 목록
│   └── 신고된 리뷰
└── 시스템 설정 (/admin/settings)
    ├── 사이트 설정
    └── 관리자 로그
```

### 1.2 화면 목록

| ID | 화면명 | 경로 | 권한 |
|---|--------|------|------|
| A-001 | 관리자 로그인 | /admin/login | Public |
| A-002 | 대시보드 | /admin/dashboard | Admin |
| A-101 | 회원 목록 | /admin/users | Admin |
| A-102 | 회원 상세 | /admin/users/:id | Admin |
| A-201 | 상품 목록 | /admin/products | Admin/Seller |
| A-202 | 상품 상세/수정 | /admin/products/:id | Admin/Seller |
| A-203 | 카테고리 관리 | /admin/categories | Admin |
| A-301 | 주문 목록 | /admin/orders | Admin/Seller |
| A-302 | 주문 상세 | /admin/orders/:id | Admin/Seller |
| A-401 | 결제 내역 | /admin/payments | Admin |
| A-501 | 리뷰 목록 | /admin/reviews | Admin |

- **시스템 알림**: 재고 부족, 신고 리뷰 등

    @wraps(f)

            return jsonify({'error': 'Admin access required'}), 403
        return f(*args, **kwargs)
    return decorated_function
