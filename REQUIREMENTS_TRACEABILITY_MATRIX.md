## RTM 개요

### 추적 항목
- **요구사항 ID** - 고유 식별자
- **요구사항명** - 기능 이름
- **우선순위** - High / Medium / Low
- **백엔드 상태** - 완료 / 진행중 / 미착수
- **프론트엔드 상태** - 완료 / 진행중 / 미착수
- **통합 테스트** - 완료 / 진행중 / 미착수
- **비고** - 특이사항, 이슈

### 상태 표기
- ✅ **완료** - 구현 및 테스트 완료
- 🚧 **진행중** - 현재 작업 중
- ⏸️ **보류** - 일시 중단
- ❌ **미착수** - 아직 시작 안 함
- ⚠️ **이슈** - 문제 발생

### 우선순위
- 🔴 **High** - 필수 기능 (MVP)
- 🟡 **Medium** - 중요 기능
- 🟢 **Low** - 부가 기능

## 요구사항 분류 체계

```
REQ-[Phase]-[Category]-[Number]

Phase:
- 01: 핵심 기능

- 03: 블록체인 시스템
- 04: NFT 시스템
- 05: 고급 평가 시스템

Category:
- AUTH: 인증/인가
- PROD: 상품 관리

- PAY: 결제 관리
- REV: 리뷰 관리
- NFT: NFT 관리

- EVAL: 평가 시스템
- ADMIN: 관리자
- UI: 사용자 인터페이스
```
